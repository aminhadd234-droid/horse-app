# Frontend (Next.js) - Example Scaffold

هذا مشروع فرونتند مبسّط مبني بـ Next.js (TypeScript).

**كيفية التشغيل محلياً (على جهازك):**
1. تأكّد من تثبيت Node.js v18+ و npm.
2. افتح سطر الأوامر في مجلد المشروع.
3. شغّل: `npm install`
4. ثم: `npm run dev`
5. افتح المتصفح عند: http://localhost:3000

**محتوى المشروع:** صفحات رئيسية، صفحة تسجيل الدخول، مكوّن Layout، وملف config بسيط.
