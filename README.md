Horse Racing Betting - Full Starter

This project is a starter fullstack app (FastAPI backend + Next.js frontend) generated by Lyra.

Quick local run (development):
1. Ensure Python 3.10+, Node 18+, Docker if desired.
2. To run with docker-compose (dev SQLite backend):
   docker-compose -f infra/docker-compose.yml up --build
3. To train ML model locally:
   cd backend/ml_model
   python train.py

Default admin credentials (dev):
  email: admin@example.com
  password: adminpass
