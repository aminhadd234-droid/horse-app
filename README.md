# Frontend Template (Next.js + TypeScript)

This is a minimal frontend template including:
- Home page (index)
- Login page
- Dashboard page
- Navbar component
- Simple bilingual setup idea (toggle)

## Run locally
1. Install Node.js (v18+) and npm.
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Notes
- The project is minimal to keep size small for a phone download.
- Add your components, styles, and translations as needed.
